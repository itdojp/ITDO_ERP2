# Frontend

ITDO ERP System v2 - Frontend Application

## Technology Stack
- React 18 + TypeScript 5
- Vite + Vitest
- Tailwind CSS
- Playwright for E2E testing

## Development
```bash
npm install
npm run dev
```

## Testing
```bash
npm test              # Unit tests
npm run test:e2e      # E2E tests
```
