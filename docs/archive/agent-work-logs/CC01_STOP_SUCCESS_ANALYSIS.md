### 運用フロー
1. **停止確認**: エージェントが実際に停止していることを確認
2. **タスク指示**: 明確で限定されたタスクを指示
3. **完了確認**: Issue #351で完了報告を確認
4. **次回準備**: 次のタスクを選定して指示
