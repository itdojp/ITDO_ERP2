## Adding Type Ignore Comments

For bulk ignoring test type annotations:
```python
# In test files
def test_something(db_session) -> None:  # type: ignore[no-untyped-def]
    pass
```

Or configure in `pyproject.toml`:
```toml
[tool.mypy]
# Existing config...

[[tool.mypy.overrides]]
module = "tests.*"
ignore_errors = true
```
